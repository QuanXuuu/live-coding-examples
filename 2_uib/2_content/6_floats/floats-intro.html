Floats

Floats sorgen dafuer, das wir elemente links oder rechts platyieren koennen.

Ein element, dem en float in css zugewiesen wurde, ist kein block-element mehr, nimmt also keinen reservierten platy mehr auf unserer seite ein.

float optionen
  • float: left; = platziert das element auf der linken seite
  • float: right; = platziert das element auf der rechten seite
  • float: none; = default, keine platzierung
