order: ;

mit order: ; koenne wir die angezeigte reihefolge von elementen veraendern, egal wie sie im HTML code sortiert sind.

der standardwert von order: ; ist 0.

1(0)
2(-1)
3(3)
4(1)
5(-2)

Wenn elemente die gleiche order:; habe, wenden sie in der reihenfolge angezeigt, in der im HTML code stehen.

1(0)
2(-1)
3(3)
4(1)
5(-2)
BOX(-1)
