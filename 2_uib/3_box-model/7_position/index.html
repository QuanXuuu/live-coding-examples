Position

Wenn wir die art der position mit position: ; definieren, können wir das element frei platzieren.

Folgende position können verändert werden, sowohl positiv, wie auch negative.

  • top = von oben nach unten
  • right = von rechts nach links
  • bottom = von unen nach oben
  • left = von links nach rechts schieben

Bei position: relative; ändern wir die position eines elements relativ zu seiner ausgangsposition.

Bei position: absolute; platzieren wir das element relative zur absoluten position des elternelemnents.

der default wert von position:; is position: static; und sagt dem element, das es ein statischer teil des html baumes ist.
