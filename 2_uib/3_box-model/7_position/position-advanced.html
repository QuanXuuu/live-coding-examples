Position Advanced

Bei position: fixed; fixieren wir ein element am broserfenster, es wird beim scrollen nicht mehr bewegt.

Bei position: sticky; fixieren wir den inhalt an der stlette die wir auf dem browserfenster wollen, sobald das element vom browser scrollen eingeholt wird.

mit z-index: ; koennen wir die position eines elements auf der tiefenachse (z) definieren, eine hoehere zahl befindet sich ueber eine niedrigeren zahl, egal ob im position oder negativen bereich. Wichtig: ein element das z-index benutzt muss eine positionierungsangabe haben.
