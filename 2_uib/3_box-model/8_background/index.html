Background Image

IMG Element

FarbBox

FarbBox mit Bild

Wenn wir in css bei Background eine url(); eingeben, koennen wir damit ein bild auf unserer seite in css verlinken.

Box mit bild

Mit background-image: url(); geben wir ein bild an, das wir verlinken wollen.

Box mit Farbe

Wenn wir eine Farbe fuer den hintergrund (von oder ohne bild) definieren wollen nutzen wir Background-color: ;

Groesse vom Bild

Mit Background-size koennen wir die groesse des bildes definieren, dies bezieht sich NICHT auf die groesse des gelandenden bildes, sondern darauf, wie gross unser bild angezeigt wird, wennn es kleiner wird als die box, wird es im standardfall wiederholt.

Der default wert von Background-size: ; gibt das bild in der groesse aus, in der es angeleft wurde.

Cover groesse

Mit Background-size: cover; wird die komplette flaeche horizonal ausgefuellt, auch wenn etwas abgeschnitten werden muss.
