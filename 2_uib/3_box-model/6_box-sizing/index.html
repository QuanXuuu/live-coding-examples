Box-sizing

box-sizing: border-box; gibt uns die möglichkeit, den border eines elements nach innen zu schieben, so nimmt der border ausserhalb des elements keinen platz weg. Der default wert ist box-sizing: content-box;

Im normfall nimmt eine box ihre zugewiesene größe ein, und fügt jeglichen padding und border aussen an das element an.
