display: inline-block

display: inline-block; sorgt dafuer, das wir block elemente nebeneinander legen koennen, alle styles wie width und height funktionieren hier immernoch.

Es gibt in css einen bug, der dafuer sorgt, das elemente die nebeneinander liegen, bei display: inline-block; ein unsichtbares leerzeichen zwischen ihnen hat, um dieses zu beheben, gibt es ein paar moeglichkeiten, die zuverlaessigste(standard) loesung, ist ,in der zeile wo ein element endet, einen kommentar anzufangen und beim naechsten element das kommentar zu beenden.
