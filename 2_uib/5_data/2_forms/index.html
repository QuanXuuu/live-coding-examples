Formular 3:

Text

das attribut placeholder="" definiert, was angezeigt werden soll, wenn das input feld leer ist.

das attribut maxlength="" definiert, wie viele zeichen in das input feld geschrieben werden duerfen.

Password

Das input feld des types "password" versteckt den geschrieben inhalt, so das er auf webseite im browser nicht sichtbar ist.

E-mail

Ein e-mail feld sollte mit den e-mail adressen regeln validieren, also von dem @ mindestens 1 buchstabe, ein @ symbol, eine domain, und eine oder mehr top-level domains.


Number

mit min und max koennen wir die kleinste und groeste auswaehlbare zahl angeben.

Range

mit value="" koennen wir angeben welchen wert der range slider zwischen 0 und 100 haben soll, wenn er geladen wird.

mit step="" koennen wir die schritte angeben, in denen der range-slider bewegt werden kann.

Textarea

ein textarea element unterscheidet sich von input-elementen, da man hier den gewuenschten text direkt zwischen die element tags platziert, nicht als value.

mit col="" geben wir an, wie breit (in buchstaben) unser textarea element sein soll, und mit rows="" geben wir an (in textzeilen) wie hoch das element sein soll.

Buttons

Ein submit button, ist dafuer da, ein formular abzuschicken.

ein reset button, ist dafuer da, ein formular zuruckzusetzen.

Nicht formular-buttons

Ein button des types "button" ist dafuer da, events in javascript auszufuehern, dafuer wird kein formular benoetigt.


Der button ueber diesem text, ist ein aequivalent zu dem folfenden button, nur das man hier den text direkt zwischen die element tags schreibt - und dieser "button" des types "button" meist in javascript ohne formular benutzt wird.

Checkbox

Wir koennen mehrere checkboxen zur auswahl stellen, der benutzer hat dann die wahl, welche er anklicken moechte.

Formularsttributen

ein feld mit dem attribut "readonly" kann nur gelesen werdenm kopiert oder selektiert werden, der inhalt kann aber nicht veraendert werden.


ein feld mit dem attribut "required" muss ausgefuellt werden, damit das formular abgeschickt werden kann.


ein feld mit dem attribut "disabled" kann selektiert werden, gelesen werden, aber er kann keine weitere interaktion durchgefuehrt werden, es erscheint grau, weil es "deaktiviert" wurde.
